Overview

This system implements a greatest prime factor finder function taken from this coding challenge

This system implements Jasmine tests. To see the tests, display the SpecRunner.html file in a browser. Here is an example of successfully invoked tests:

Usage

Simply download the files and invoke the primeFactors() function. You can also simply run index.html file. See primefactors.js for detail.

Here is an example call to the function:

<script>
console.log("The answer should be 6857: output = " + primeFactors(600851475143));
</script>
